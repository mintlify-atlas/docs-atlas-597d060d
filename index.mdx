Bienvenido al repositorio oficial del curso **Plataformas Abiertas** (2026, 2do cuatrimestre). Aquí encontrarás todas las guías, ejemplos de código y materiales del curso, organizados por tema. El curso cubre control de versiones con Git y GitHub, bases de datos NoSQL con MongoDB, y construcción de APIs REST con Python y Flask.

## Temario del Curso

<Steps>
  <Step title="Semanas 1–5: Fundamentos">
    Control de versiones con Git y GitHub, bases de datos NoSQL con MongoDB Atlas, consultas CRUD y agregaciones.
  </Step>
  <Step title="Semanas 6–15: Desarrollo Backend">
    Introducción a Python, programación orientada a objetos y construcción de APIs REST con Flask y Flask-CORS.
  </Step>
  <Step title="Proyecto integrador">
    Aplicación completa con API Flask conectada a MongoDB, publicada en GitHub y colaboración mediante Pull Requests.
  </Step>
</Steps>
